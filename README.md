## 🧩 Extensiones y Gestores de Referencias

### <img src="https://lh3.googleusercontent.com/OapR8jWa1sKFMFdtYeHigBP_Z76FU7jadrDYMyS-SEzvwVwMgKEGuwowFTDyw7s3D9TZ2UszjGosNgGeqXvodR20=s60" width="24" height="24"> Diigo Web Collector

**Descripción:** Herramienta para guardar, etiquetar y anotar páginas web. Facilita la gestión y organización de fuentes.

**Instrucciones de Uso:**
- 🧩 En caso tenga extensión Chrome instalada, se solicitará Login y Password o cuenta Google
- 🔑 **Si tiene la extensión instalada:** Se copiará el link de acceso, seleccione el link y presione enter
- 🛠 **Si no tiene la extensión instalada:** Hacer click en el botón instalación

[![Instalar](https://img.shields.io/badge/Chrome-Instalar_Extensión-blue?style=flat-square)](https://chrome.google.com/webstore/detail/diigo-web-collector/oojbgadfejifecebmdnhhkbhdjaphole)

### <img src="https://lh3.googleusercontent.com/ipkToqos1ZNXIZSAUMGYbNgdK0t05bnSTmUaOmwDsKPAW8zAfOhmpGJxBfcYPX-HVTiAzOrjRMHG6FPxmo_-84gl2Q=s60" width="24" height="24"> Science Research Assistant

**Instrucciones de Uso:**

[![Instalar](https://img.shields.io/badge/Chrome-Instalar_Extensión-blue?style=flat-square)](https://chrome.google.com/webstore/detail/science-research-assistant/ceacgaccjcomdbnoodjpllihjmeflfmg)

---

### <img src="https://upload.wikimedia.org/wikipedia/commons/3/3e/Unpaywall_logo_2017.svg" width="24" height="24"> Unpaywall

**Descripción:** Permite acceder de forma legal a versiones gratuitas de artículos académicos desde sitios oficiales.

**Instrucciones de Uso:**
- 📖 Se abrirá una página de búsqueda de artículos (Nature)
- 📥 En el lado derecho dar clic al **candado verde** para descargar el artículo
- ✖ Si el candado de la derecha es de color **gris**, NO podrá descargar

[![Instalar](https://img.shields.io/badge/Chrome-Instalar_Extensión-blue?style=flat-square)](https://unpaywall.org/products/extension)
